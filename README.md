# BitView
A simple bit-viewer with support for:
* Custom offsets in file
* Custom bit size
* Custom bit colors
* Custom width
* Grid-mode toggle
* Bit-shifts, rev-8

## Usage
Build (.NET core is required) or download one of the self-contained executables [here](https://github.com/olegzl/bitview/releases).
